基于 [Vue.js](https://cn.vuejs.org/index.html), [Vue CLI](https://cli.vuejs.org/zh/), [即刻喵 1.x](https://github.com/coder-ysj/jike-meow) 重写.

## 主要功能

- 👤 网页登录 - 为网页端提供一键登录功能
- 👥 多人管理 - 可同时登录最多三个账号
- 🔔 消息列表 - 直接查看详细的通知内容
- 🔒 保护隐私 - 不收集任何个人信息
